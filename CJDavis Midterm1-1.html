Q2. Describing your intuitions about sets in words [10 pts]¶

The null set is an empty set, with no elements. The null set can be written $\emptyset$. Thus, $\emptyset = \{ \}$

Say that you have sets $A$ and $B$, where $A \subset B$ and where $ \vert A \vert > 0$. Is it the case that $ \vert B \vert > \vert \emptyset \vert $?

Answer yes or no. Explain why it is or is not the case that $\vert B \vert > \vert \emptyset \vert$ using a few short English sentences that clearly and convincingly explain your answer. Pretend like you are explaining to a fellow classmate who is confused about this.

You may want to use a picture or diagrams to support your argument. You can upload any diagrams that you draw to help with your explanation to Canvas. Just be clear to be clear about which diagram corresponds to Q2.

$A$ is a subset of $B$ and $A$ has 1 or more elements. If $A$ is a "proper subset," then $B$ must have at least one more element than $A$. Cardinality is similar to absolute value in that you must have a positive number, or zero. In this case, it is a positive number of elements, or zero. What is the cardinality of a null or empty set? The number of elements in the set is zero, so the cardinality has to be zero. Therfore the cardinality of $\vert B\vert$ must be greater than $\vert\emptyset\vert$. It is also informative to consider that $\vert B\vert$ has to be greater than $\vert A\vert$ which is itself $\vert A\vert>0$. The only odd case that is possible (but not in this example), is $A\subseteq B$ where $\vert B\vert$ might = $\vert A\vert$.

Q4. Max norm [10 pts]¶

The max norm of a vector $\boldsymbol{x}$ is defined as the largest element in $\boldsymbol{x}$, by absolute value.

For instance, the max norm of $\boldsymbol{x}=<4,-7>$ is 7 because $\vert \boldsymbol{x}_2 \vert =7$ and $\boldsymbol{x}_2$ has the highest absolute value from among components of $\boldsymbol{x}$.

The max norm of $\boldsymbol{x}=<12,-2,44>$ is 44.

More formally, we can define

maxnorm($\boldsymbol{x}$) = max$(\vert \boldsymbol{x}_1 \vert, \vert \boldsymbol{x}_2 \vert ... \vert \boldsymbol{x}_N \vert)$ where max returns the highest input and $\vert \boldsymbol{x}_i \vert$ is the absolute value of component $i$ of $\boldsymbol{x}$.

Implement the maxnorm function in midterm_1_part2.py and turn in this file along with your midterm.

def max_norm(x):
    iMax = 0
    for i in x:
        if iMax < abs(i):  # we know we have a higher absolute value
            iMax = abs(i)  # set the iMax value to it
    return iMax            # we are done with the loop.  return what we have.
